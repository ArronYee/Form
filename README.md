## Install

In your project build.gradle:

```sh
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

In your needed module
```sh
	dependencies {
		  implementation 'com.github.ArronYee:Form:0.0.3'
	}
```
